[Coding Practice] 명예의 전당 (1)

명예의 전당 (1) 문제에 대한 설명은 프로그래머스 사이트를 참조바랍니다.

이 문제의 요구사항은 정해진 리스트의 크기만큼만 점수를 저장 가능하고 그 중에서 최저값을 answer에 저장하는 것입니다. 요소의 정렬 및 삭제를 용이하게 하기 위해서 ArrayList를 사용하였고, hallOfFame 리스트의 크기가 k보다 커졌을 때 바로 제일 작은 크기의 요소를 제거해주면 되는데 내림차순이 되어있으므로 맨 마지막 인덱스의 값을 제거해주면 됩니다.

import java.util.*;

class Solution {
    public int[] solution(int k, int[] score) {
        // answer는 score의 길이만큼 설정.
        int[] answer = new int[score.length];
        
        // 명예의 전당 리스트에 최대 k개의 요소가 들어간다.
        // 그 중에서 매일 최저점을 answer에 저장.
        // 명예의 전당은 ArrayList로 저장하고 내림차순으로 정렬.
        // k개가 되었을 때부터 내림차순으로 정렬 후 제일 작은 것을 제거.
        ArrayList<Integer> hallOfFame = new ArrayList<>();
        
        for (int i = 0; i < score.length; i++) {
            hallOfFame.add(score[i]);
            Collections.sort(hallOfFame, Collections.reverseOrder());
            if (hallOfFame.size() <= k) {
                answer[i] = hallOfFame.get(hallOfFame.size() - 1);
            }
            else {
                hallOfFame.remove(hallOfFame.size() - 1);
                answer[i] = hallOfFame.get(hallOfFame.size() - 1);
            }
        }
        
        // PriorityQueue를 이용하여 자동 오름차순 정렬 및 peek()로 값 저장하는 방법도 가능.
        return answer;
    }
}

풀기에는 무난한 문제였지만 다른 풀이를 보니 애초에 PriorityQueue를 이용하여 자동 오름차순 정렬을 시키는 방법을 사용한 것을 확인하여 조금 더 간략하게 코드를 짤 수 있다는 것을 알게되었습니다.
