[Coding Practice] 체육복

체육복 문제에 대한 설명은 프로그래머스 사이트를 참조바랍니다.

주석에 설명되어있지만 이 문제는 lost에 있는 요소를 최대한 많이 줄여야 하기 때문에 lost 요소 개수를 줄여나가기만 하면 최적의 해를 구할 수 있습니다. 그러나 그 전에 먼저 예비 체육복을 가지고 있어도 그것이 없어졌다면 어차피 옆사람에게 빌려줄 수가 없으므로 lost와 reserve에 중복되는 요소들이 있다면 미리 제거를 해줍니다. 학생 수가 최대 30명 정도이기 때문에 이 경우에는 배열을 하나 만들어 각 인덱스가 학생 번호를 나타낼 수 있도록 해주는 것이 좋습니다. 여기서는 양 옆으로 각각 버퍼를 한 개씩 두어서 배열의 크기가 N + 2가 되었는데, 양 옆을 비교하면서 학생 전체를 순회하려면 버퍼가 있어야 반복문 인덱스 에러가 발생하지 않기 때문에 설정한 것입니다.

중복 제거가 완료된 후에는 해당 배열을 반복문으로 순회하여 체육복의 개수가 0개인 경우 양 옆에서 1개를 가지고 오도록 코드를 만들었고, 다시 반복문을 돌리지 않기 위해 바로 해당 인덱스에 1 이상의 수가 있으면 answer에 1씩 추가하도록 하였습니다.

class Solution {
    public int solution(int n, int[] lost, int[] reserve) {
        int answer = 0;
        // lost를 최대한 많이 없애야 함.
        // lost + 1 or -1 check
        // 먼저 lost와 reserve에서 중복되는 요소들을 제거한다.
        // 2명 이상 30명 이하이므로 인덱스가 학생 번호이고,
        // 체육복의 개수를 요소로 하는 배열 생성.
        // n + 2 크기로 해서 앞뒤를 버퍼로 설정.
        int[] numSportswear = new int[n + 2];
        for (int i = 1; i <= n; i++) {
            numSportswear[i] = 1;
        }
        // lost의 경우 해당 요소를 -1.
        for (Integer l : lost) {
            numSportswear[l]--;
        }
        // reserve의 경우 해당 요소를 +1.
        for (Integer r : reserve) {
            numSportswear[r]++;
        }
        
        // 반복문으로 순회하여 값이 0인 경우 해당 인덱스 -1이나 +1의 값을 1만큼 가져온다.
        for (int j = 1; j <= n; j++) {
            if (numSportswear[j] == 0) {
                if (numSportswear[j - 1] >= 2) {
                    numSportswear[j]++;
                    numSportswear[j - 1]--;
                }
                else if (numSportswear[j + 1] >= 2) {
                    numSportswear[j]++;
                    numSportswear[j + 1]--;
                }
            }
            
            // 체육복을 빌려준 후 바로 체육복을 입을 수 있는 사람 수에 더해준다.
            if (numSportswear[j] >= 1) {
                answer++;
            }
        }

        return answer;
    }
}

코드 제출 후 다른 풀이와 비교하였더니 비슷한 컨셉으로 작성된 것이 있었습니다. 다른 사람들 중에도 비슷하게 코드를 짠 사람이 있다는 것을 알게되니 시간을 써서 열심히 작성한 것에 대한 보람을 느낄 수 있었고, 나름 효율적인 코드가 만들어진 것 같은 느낌도 들었습니다.
